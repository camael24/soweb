Public
__call ( $name, $arguments)

Helper for adding rules. Methods are concatenated by _. If prefixed by _, it's a private rule. In addition, the keyword “any” takes place for all methods. Examples: get(…) : addRule(…, array('get'), …); get_post(…) : addRule(…, array('get', 'post'), …); post_get(…) : same that above; _get(…) : addPrivateRule(…, array('get'), …); any(…) : addRule(…, array(), …); head_delete(…): addRule(…, array('head', 'delete'), …).
